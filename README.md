# HealthScope 🏥

HealthScope is a healthcare-based web application developed using Django. It helps users get preliminary health guidance based on symptoms through an intelligent chatbot system. The project also includes user authentication and chat history features.

## 📌 Project Objective

The objective of HealthScope is to provide users with a basic healthcare assistance platform where they can enter symptoms and receive possible disease-related guidance.

This project is developed for educational purposes as a first-year college capstone project.

---

## 🚀 Features

- User Registration and Login System
- Health Chatbot Assistance
- Symptom-Based Disease Suggestions
- Chat History Storage
- User Authentication
- Interactive User Interface
- Database Integration

## 💡 How It Works

1. User creates an account or logs in.
2. User enters symptoms in the chatbot.
3. The system compares symptoms with predefined disease data.
4. Relevant disease guidance is shown.
5. Chat history is stored for future reference.
